Baten RA, Bagley D, Tenesaca A, Clark F, Bagrow JP, Ghoshal G, Hoque ME, Creativity in temporal social networks: How divergent thinking is impacted by one's choice of peers. _Journal of the Royal Society Interface_, (accepted; arXiv preprint arXiv:1911.11395), September 2020
